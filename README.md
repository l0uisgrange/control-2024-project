# Mastermind

This is the spring 2024 semester project for the _Microcontrollers_ course at EPFL from [Schmid Alexandre](https://people.epfl.ch/alexandre.schmid?lang=en). We had to implement a library in Assembly that manages the communication between the different hardware parts of our microcontroller. Our final result was a number-guessing game.
